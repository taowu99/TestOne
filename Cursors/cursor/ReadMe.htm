You may use these cursors for free. But please include this small text file and inform me, if you want to distribute them. Thanks.

The cursors (pointers) should be copied to the windows/cursors directory. If you launch mouse from control panel and change a pointer, the default directory where pointers are located is the cursors dir in windows. You must set your Desktop to more than 256 colors to view the cursors in full color.
